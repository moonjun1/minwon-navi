# 서비스 개발 기획서

## 1. 서비스명
민원 내비 - AI 민원 안내 서비스

## 2. 서비스 개요
국민이 민원 업무를 처리하고자 할 때, AI 챗봇을 통해 필요 서류, 담당 부서, 절차를 안내받고, 실시간 민원실 대기현황 데이터를 기반으로 가장 한산한 민원실을 추천받을 수 있는 서비스입니다.

## 3. 활용 데이터
### 필수 활용 데이터 (전국 통합개방데이터)
- **민원실 이용 현황 실시간 정보 (2종)**
  - 민원실 기본정보 (cso_info_v2): 민원실명, 주소, 위경도, 운영시간, 야간/주말 운영 여부
  - 민원실 실시간 대기현황 (cso_realtime_v2): 업무별 대기인수, 호출창구번호
- **초정밀버스 위치 실시간 정보 (3종)**
  - 노선 기본정보 (mst_info): 노선번호, 기종점
  - 노선 경유지 정보 (ps_info): 정류소명, 위경도
  - 버스 실시간 위치 (rtm_loc_info): GPS 위치, 속도

### 추가 활용 기술
- OpenAI GPT-4o-mini: 자연어 대화 기반 민원 안내
- Leaflet: 민원실 위치 지도 시각화
- Geolocation API: 사용자 위치 기반 주변 민원실 추천

## 4. 서비스 주요 기능

### 4-1. AI 민원 안내 챗봇
- 사용자가 자연어로 민원 관련 질문 ("전입신고 하려는데", "여권 발급 방법")
- AI가 민원 종류 파악 → 필요 서류, 담당 부서, 처리 절차 안내
- 실시간 민원실 대기현황 API 연동하여 가장 한산한 민원실 추천

### 4-2. 실시간 민원실 현황 대시보드
- 전국 민원실의 실시간 대기인수를 카드 형태로 표시
- 사용자 위치 기반 가까운 민원실 순 정렬 (거리 표시)
- 30초 간격 자동 갱신, 수동 새로고침 지원

### 4-3. 민원실 위치 지도
- Leaflet 기반 지도에 민원실 위치 마커 표시
- 대기인수에 따라 마커 색상 변경 (0명: 초록, 1-3명: 노랑, 4명+: 빨강)
- 마커 클릭 시 민원실 정보 팝업

### 4-4. 길찾기 연동
- 각 민원실 카드에 네이버맵/카카오맵 길찾기 버튼
- 클릭 시 해당 지도앱의 길찾기 화면으로 바로 이동

### 4-5. 추천 질문
- 자주 묻는 민원 질문을 칩 버튼으로 제공
- 전입신고, 여권 발급, 주민등록등본, 인감증명서, 운전면허 갱신

## 5. 기대효과
- **국민 편익**: 민원실 방문 전 대기 현황 확인으로 불필요한 대기시간 절감
- **AI 활용**: 자연어 대화로 복잡한 민원 절차를 쉽게 안내
- **데이터 가치**: 실시간 대기현황 데이터를 국민이 직접 활용할 수 있는 서비스로 전환
- **확산 가능성**: 전국 모든 민원실 대상, 별도 설치 없이 웹으로 접근 가능

## 6. 기술 스택
- Frontend: Next.js 16, React, TypeScript, Tailwind CSS, shadcn/ui
- AI: OpenAI GPT-4o-mini (Function Calling)
- 지도: Leaflet (오픈소스)
- 배포: Vercel
- 공공데이터: 행정안전부 전국 통합개방데이터 API
